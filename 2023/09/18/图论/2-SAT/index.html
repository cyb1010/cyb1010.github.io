2-SAT

什么是 2-SAT ？

SAT 是 Satisfiability 的缩写，意为可满足性。
n-SAT 问题即对一串 bool 变量进行赋值，满足 n 元 bool 方程组。
当 n≥3n\ge3n≥3 时，n-SAT 问题已被证明是NP完全的，暂时没有多项式复杂度的做法；
但今天的主角 2-SAT 是可以在线性时间复杂度内解决的。

如何求解 2-SAT 问题？

转化为图论问题。
具体的，对于每个变量 xxx ，我们建立两个点 xxx 与 ¬x\neg x¬x ，分别表示 xxx 取 true 或 false 。对于每个要求 (a∨b)(a\vee b)(a∨b) ，连接 ¬a→b\neg a \to b¬a→b 和 ¬b→a\neg b\to a¬b→a ，表示“ aaa 为假 bbb 必为真，bbb 为假 aaa 必为真”。

建好图后，我们发现若点 xxx 为真，则点 xxx 可到达的点必然都为真。显然的，若点 xxx 与 ¬x\neg x¬x 可互达，此问题必然无解，毕竟一个值不可能同时即为真又为假 薛定谔的bool（不是）\tiny{\text{薛定谔的bool（不是）}}薛定谔的bool（不是） 。稍作思考后，我们发现同一 强连通分量 内的点的值必然相等，因为均可互达。
